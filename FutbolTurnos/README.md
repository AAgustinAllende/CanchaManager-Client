# Sistema de Autogestión de Turnos para Canchas de Fútbol

Aplicación web orientada al cliente final, que permite reservar turnos en canchas de fútbol de forma simple e intuitiva, desde el login hasta el pago y la confirmación de la reserva.
El proyecto está enfocado en la experiencia del usuario, el flujo completo de reserva y la integración de servicios comunes en aplicaciones reales.

# Funcionalidades principales

-Autenticación de usuarios
-Login con email y contraseña
-Autenticación con Google

# Reserva de turnos

-Selección de cancha
-Selección de fecha y horario
-Formulario con datos del usuario

# Checkout y pago (demo)

-Pasarela de pago ficticia
-Confirmación de la operación

# Generación de QR

-Código QR con los datos de la reserva


# Historial de turnos

-Visualización de reservas realizadas


# Aspectos técnicos

Frontend desarrollado con:

-React
-CSS
-Tailwind CSS
-Autenticación y flujo de pago implementados del lado del frontend con fines demostrativos.
-Generación dinámica de QR con los datos de la reserva.

# backend

El sistema cuenta con un backend desarrollado en el mismo stack utilizado en otros módulos del proyecto, el cual funciona en entorno local.


# Objetivo del proyecto

Este proyecto fue desarrollado con fines educativos y de portfolio, con el objetivo de:

-Simular un sistema real de reservas
-Practicar flujos completos de usuario
-Integrar autenticación, pagos y confirmaciones
